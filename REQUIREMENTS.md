# Requirements: How I FIREd Myself Website

## Project Overview
- **Project name**: How I FIREd Myself Website
- **Type**: Marketing website + blog for retirement book
- **Target users**: 50-65 year old pre-retirees
- **Goal**: Book promotion, lead capture, blog content delivery

## Tech Stack
- Framework: Next.js + Tailwind CSS
- Deployment: GitHub Pages
- Custom Domain: howifiredmyself.com
- Database: Supabase (leads table)

## Pages Required

### 1. Homepage (/)
- Hero section with book tagline
- Problem/Solution sections
- "What's Inside the Book" overview
- Lead magnet capture form
- Testimonials
- Footer with links

### 2. Blog (/blog)
- List of 10+ SEO-optimized blog posts
- Each post with meta tags

### 3. Blog Post Template (/blog/[slug])
- SEO-optimized article
- Related posts
- Share buttons

### 4. About (/about)
- Author bio
- Book backstory

### 5. FAQ (/faq)
- Comprehensive FAQ for SEO
- Schema markup for rich results

### 6. Lead Magnet (/jumpstart)
- Download page for 30-Day Guide
- Email capture

### 7. Thank You (/thank-you)
- After email signup
- Download link

## Features

### Email Capture
- Form fields: name, email
- Store in Supabase leads table

### SEO
- Meta titles/descriptions on every page
- Schema markup: Book, FAQ, Article
- Open Graph tags
- Sitemap
- robots.txt

### Design
- Trustworthy, professional for 50-65 demographic
- Mobile responsive

## Supabase Schema
```sql
CREATE TABLE leads (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  email TEXT UNIQUE NOT NULL,
  name TEXT,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  source TEXT DEFAULT 'website'
);
```
